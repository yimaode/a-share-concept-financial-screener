# 贡献指南

感谢参与“基于投资概念固化的 A 股财报量化筛选器”。

## 开发流程

1. 从最新默认分支创建功能分支。
2. 安装开发依赖：`pip install -e '.[dev]'`。
3. 修改代码并添加覆盖真实失败场景的测试。
4. 运行 `python -m pytest -q`。
5. 检查暂存内容中没有私有语料、财报副本、Cookie、密钥、持仓或自选股。
6. 提交小而清晰的 Pull Request，说明行为变化、测试和数据来源。

## 数据与版权

不要提交无权再分发的文章、课程、聊天记录、付费研究、完整财报副本或第三方数据库导出。测试优先使用最小化合成夹具；展示数据只能使用公开来源的聚合结果，并标明时间和来源。

## 代码要求

- Python 3.11+。
- 路径使用 `pathlib.Path`。
- 失败必须显式返回，不得用空 `except` 隐藏错误。
- 财务数字和 PDF 语言证据保持分离。
- 新外部接口必须有超时、重试、错误说明和缺失数据状态。
- 修改评分规则时必须补充边界值和缺失数据测试。
